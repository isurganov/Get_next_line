<!-- GETTING STARTED -->
## Getting Started

First of all clone my repo with ```git clone```.

### Run it

Some steps how to run this lib.
* run Makefile
  ```sh
  make 
  ```
* clean obj files
  ```sh
  make clean
  ```
* clean obj + get_next_line.a
  ```sh
  make fclean
  ```
* fclean all
  ```sh
  make re
  ```
